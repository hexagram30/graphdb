# hexagram30/graphdb

*A graph database, built on OrientDB, for use by hexagram30 projects*

[![][logo]][logo-large]


## Usage

Standalone:

```
$ lein docker-start
```

For use as part of a component-based system, see
[hxgm30.mush.components.database][comp-graphdb].
